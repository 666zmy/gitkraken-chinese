控制台
使用说明 对于有"有道翻译"API的用户:
  1. 将旧版汉化文件(如 strings_9.5.1.json )重命名为 strings_old.json
  2. 使
    • compare.html;
    • strings_old.json;
    • 更新的英文 strings.json
    位于同级目录
  3. 使用 VSCode 的 Live Server 插件打开 compare.html ,将自动解析两个json文件
  4. 填写有道API并点击 保存
  5. 点击 对比 按钮得到差异内容
  6. 点击 自动翻译 按钮翻译,并将结果使用 格式化工具 整理
  7. 将整理好后的内容覆盖 strings.json 文件
  8. 重启 GitKraken
对于没有API的用户
  1. 前三个步骤和上面的一样
  2. 点击 对比 按钮得到差异内容
  3. 点击 自动生成json 按钮生成整合后的 json 文件
  4. 复制并翻译文本框内的文件
  5. 将翻译好的内容使用 格式化工具 整理
  6. 承接上面的第七步
有关有道API的申请步骤,参阅 关于有道API
